# Amjad's Portfolio

A modern, interactive portfolio website built with React and TypeScript featuring stunning visual effects and smooth animations.

## Features

- **Dynamic Mouse Effects**: Beautiful rainbow-colored particle system and lighting effects that respond to mouse movement
- **3D Graphics**: Interactive 3D shapes rendered with Three.js that react to user interaction
- **Gradient Mesh Background**: Animated gradient blobs create a dynamic, fluid background
- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern Tech Stack**: Built with React, TypeScript, Vite, and React Three Fiber
- **Smooth Animations**: Framer Motion powers smooth page transitions and scroll animations

## Projects Showcase

The portfolio includes featured projects:
- **Glint3D**: Lightweight 3D rendering engine with cross-platform support
- **AI Debugger**: Advanced C++ debugging tool using LLVM and machine learning
- **Opsmind**: Production-grade AI-powered incident management system

## Tech Stack

- React 18
- TypeScript
- Vite
- React Three Fiber (@react-three/fiber)
- Framer Motion
- CSS3 with modern animations
- Canvas API for custom graphics

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
